# Security Policy

## Supported Versions

This repository is currently pre-1.0. Security fixes will be applied to the latest release branch.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory or contacting the maintainer listed in the repository profile.

Do not include secrets, private PDFs, access tokens, personal data, or proprietary data in public issues.

## Data Boundary

This project is intended for public or explicitly authorized sources only. It must not be used for unauthorized data access, credential collection, evasion, persistence, or extraction of private information.
