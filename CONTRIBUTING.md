# Contributing

Contributions are welcome if they improve reproducibility, evidence tracking, extraction quality, or documentation.

## Good First Contributions

- Add public fixture documents.
- Improve extraction schema examples.
- Add tests for evidence-linked answers.
- Improve documentation.
- Add evaluation metrics.

## Rules

- Use only public or explicitly authorized sources.
- Do not commit credentials, API keys, private data, or copyrighted full-text PDFs.
- Keep claims tied to evidence.
- Add tests for behavior changes.
