# Maintainer Statement

Maintainer: Ranxing Chu

Role: Primary maintainer

I am the primary maintainer of Materials Evidence Stack. I created the initial evidence model, demo retrieval flow, tests, documentation, and release plan. The project is designed as a small, reproducible open-source seed for evidence-grounded materials AI, where generated answers preserve explicit links to source documents and evidence spans.

My research context is PhD-level materials science and AI-assisted materials research at Harbin Institute of Technology, with emphasis on titanium alloys, titanium matrix composites, L-DED process-structure-property evidence chains, and source-grounded knowledge workflows.

This repository is the public, minimal seed of a broader local/private research workflow background. That background includes TiKB / TiForge-style schema design, large-scale literature evidence handling, provenance tracking, field lineage, quality gates, uncertainty labels, unit checks, and audit records for titanium materials research. The public repository deliberately reduces that broader experience to a small fixture, clear evidence model, and reproducible tests.

The maintenance plan is to keep the repository public, reproducible, and auditable:

- Maintain source-bound demo behavior.
- Add public fixtures only, avoiding private PDFs, paid data, unauthorized scraping, and credentials.
- Track roadmap work through public GitHub issues.
- Release small, testable versions with clear changelog entries.
- Use Codex/OpenAI assistance for tests, documentation, issue triage, release notes, and evidence-linking quality checks.

The project should not claim validated scientific facts beyond the evidence packaged in each demo or fixture.
