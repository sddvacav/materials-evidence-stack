# Evaluation

Initial metrics:

- Evidence coverage: percentage of answers with a non-empty evidence link.
- Citation fidelity: whether the answer is supported by the evidence text.
- Numeric accuracy: whether extracted values and units match the source.
- Field completeness: whether material, process, property, condition, and source fields are populated.

Future work:

- Add fixture-based extraction benchmarks.
- Add human review workflow.
- Add retrieval metrics such as Recall@k and NDCG@10.
